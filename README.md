# 경기도 고등학교 찾기

경기도 고등학교를 쉽게 검색하고 확인할 수 있는 단순한 공공데이터 기반 사이트입니다.

## 핵심 기능
- 학교명, 시군구, 설립 구분으로 검색
- 목록 카드와 상세 보기 동시 제공
- `고등학교현황.xls` 같은 공공데이터 파일을 브라우저에서 직접 업로드해 사용
- GitHub Pages 자동 배포

## 사용 방법
1. `index.html`을 연다.
2. `고등학교현황.xls` 또는 `xlsx` 파일을 업로드한다.
3. 검색어와 필터로 원하는 학교를 찾는다.
4. 카드의 `상세 페이지` 링크 또는 목록 선택으로 자세히 본다.

## 데이터 원칙
- 공개 가능한 학교 정보만 사용한다.
- 학생 개인정보, 연락처, 내부 행정정보는 저장하거나 업로드하지 않는다.
- 원본 엑셀 파일은 저장소에 올리지 않는다.
- 공개 배포용 데이터는 `data/schools.json`처럼 정제된 형태만 둔다.

## GitHub Pages
- `main` 브랜치로 푸시하면 `.github/workflows/deploy.yml`이 자동 배포한다.
- 정적 파일만 사용하므로 별도 빌드 도구가 필요 없다.

## 데이터 형식
`data/schools.json`에는 다음 필드가 있으면 가장 잘 동작합니다.
- `schoolName`
- `address`
- `district`
- `schoolType`
- `founding`
- `phone`
- `homepage`
- `note`

원본 공공데이터 열 이름이 달라도, 업로드 시 자동으로 최대한 추정해서 읽도록 설계되어 있습니다.
